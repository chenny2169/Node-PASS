## 環境設定
1.目前我們採用`pm2`來幫我們監控並開啟程式，請在`npm start`前先下`npm install pm2 -g`  
  官方文件:http://pm2.keymetrics.io/docs/usage/quick-start/

2.為了讓專案在下`npm start`時可以自動連上Mongodb，請把 MongoDB 安装路径的 bin 路徑放到環境變數
